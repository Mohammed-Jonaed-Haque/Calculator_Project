# Python Calculator 🧮

## Overview
This project is a **command-line calculator built in Python** that allows users to perform arithmetic operations such as addition, subtraction, multiplication, and division. The program takes user input, performs the selected operation, and displays the result in the terminal.

---

## Features
- Perform arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Uses functions for each operation
- Dictionary-based operation selection
- Interactive command-line interface
- Allows users to continue calculations with previous results


---

## Technologies Used
- **Python**
- Programming concepts used:
  - Functions
  - Dictionaries
  - Loops
  - User input
  - Conditional statements

---

## How It Works
1. The program asks the user to enter the **first number**.
2. The user selects an **operation** (`+`, `-`, `*`, `/`).
3. The user enters the **second number**.
4. The calculator performs the operation using the corresponding function.
5. The result is displayed.
6. The user can continue calculating with the result or start a new calculation.

---

## Example 

```
Enter the first number: 10
Choose operation: +
Enter the second number: 5

Result: 10 + 5 = 15
```
